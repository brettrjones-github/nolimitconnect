=== GoTvPtoP Prerequisites ====
1.) Python 2.7.13 or Python 2.7.14
	 NOTE: Unless you want to change Paths in visual studio then python must be installed to default location C:\Python27
2.) YASM 
3.) visual studio 2015 ( Community edition is fine )
4.) Qt 5.9 ( Tested with Qt 5.9.6 but other versions may work )
5.) Java Oracle JDK/JRE 1.8.0_131 
6.) MingWin is NOT required.. even for ffmpeg build ( have removed all dependecy on migwin and made substitute functions )
