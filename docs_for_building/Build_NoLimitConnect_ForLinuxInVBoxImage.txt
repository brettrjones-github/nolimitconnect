=== Build No Limit Connect For Linux in Virtual Box Image ===
1.) Follow the instructions in
        Create_Linux_Virtual_Image_With_Oracle_VirtualBoxMint20_3.txt
2.) Open console
3.) cd ~/nolimitdev/nolimitconnect
4.) git pull (to update if desired)
5.) Open Qt Creator
6.) Rebuild by build number BuildNlc1_NlcDependLibs, BuildNlc2_NlcCore and BuildNlc3_NlcAppOnly
7.) If building for Ubuntu you may need additional library(s)
    apt-get install libxext-dev
